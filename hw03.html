3.4 Problem 4: Random Circles in Grid

Write a function random_circles to create a shape with many random concentric circles and use that to put it in a grid. You can use the beside, below and grid function defined in the last class. They are included below for your convenience.

def beside(s1, s2):
    s1 = s1 | scale(x=0.5) | translate(x=-75)
    s2 = s2 | scale(x=0.5) | translate(x=75)    
    return s1+s2

def below(s1, s2):
    s1 = s1 | scale(y=0.5) | translate(y=75)
    s2 = s2 | scale(y=0.5) | translate(y=-75)    
    return s1+s2

def grid(s1, s2, s3, s4):
    return below(
        beside(s1, s2),
        beside(s3, s4))
def random_circles(r, n):
    shapes = [circle(r=random(r)) for i in range(n)]
    return combine(shapes)
def repeat4(shape):
    return grid(shape, shape, shape, shape)
grid(random_circles(100, 25), random_circles(100, 25), random_circles(100, 25), random_circles(100, 25))
